About CarFinder

CarFinder is a demo front-end project that helps users choose cars by answering a short interactive questionnaire. It's built with simple, portable HTML/CSS/JavaScript and contains a sample dataset of 120 cars (images are placeholders).

Features

  • 7-question animated wizard
  • Spinning tyre loader
  • Searchable results
  • 100+ car data entries
  • Car comparison tool
  • Wishlist functionality
  • Interactive car game
